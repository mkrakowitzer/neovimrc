# neovimrc files

Heavily based off the primeagens neovimrc with my own customisations.

## Telescope

| mapping       | description                         |
| ------------- | ----------------------------------- |
| <leader>pv    | open file tree                      |
| <leader>pf    | find file                           |
| <C-p>         | find file (git files only)          |
| <leader>ps    | grep for term in files              |

## harpoon

| mapping       | description                         |
| ------------- | ----------------------------------- |
| <C-e>         | open harpoon                        |
| <leader>a     | add file to harpoon                 |
| <C-h>         | next file                           |
| <C-l>         | previous file                       |

## undo

| mapping       | description                         |
| ------------- | ----------------------------------- |
| <leader>u     | undo                                |
| ctrl-w        | switch panes                        |

## fugitive

| mapping       | description                         |
| ------------- | ----------------------------------- |
| <leader>gs    | git status                          |
